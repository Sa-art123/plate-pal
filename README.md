# plate-pal
ML-based food calorie estimation system using YOLOv8 and PyQt5

plate-pal is a desktop application that estimates food calories from images using a YOLOv8-based model and provides BMI-based health recommendations.

 Features
- Food detection from images
- Calorie estimation
- BMI calculation
- User-friendly PyQt5 interface

 Tech Stack
- Python
- YOLOv8
- PyQt5

 How to Run
1. Install required libraries:
   pip install -r requirements.txt
2. Run the application:
   python main.py

 Note
Model weights and datasets are not included due to size constraints.
